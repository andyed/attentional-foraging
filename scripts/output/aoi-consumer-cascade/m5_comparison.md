# M5 cursor-only classifier — absolute vs organic-bbox attribution

Generated by `scripts/m5_cursor_only_taxonomy.py --attribution {absolute,organic}` on 2026-05-02. Model: `LogisticRegression(class_weight='balanced', C=1.0)` on standardized M4 cursor features. Protocol: GroupKFold-by-participant, 47 folds (LOSO).

**Important.** This is a side-by-side of two different cohorts, not the same cohort scored two ways. Under organic attribution the population (`approached & ~was_clicked`) shrinks 2,355 → 2,070 records, and the class balance shifts. The headline AUC numbers are therefore not directly comparable as a "before/after" — read them as "the [LAB]→[WILD] bootstrap behaviour under each attribution method, on its own native cohort."

## Population shifts

| | Absolute | Organic | Δ |
|---|---|---|---|
| Approached non-click episodes | 2,355 | **2,070** | −285 (−12.1%) |
| Deferred (NB22 gaze_regression=True) | 1,916 (81.4%) | **1,557 (75.2%)** | −359 records, share −6.2 pp |
| Evaluated-rejected | 439 (18.6%) | **513 (24.8%)** | +74 records, share +6.2 pp |
| LOSO groups (participants) with both classes | 47 | **46** | −1 (one participant's eval-rej class emptied under bbox) |

The eval-rejected class **grows in absolute count** under bbox even though the cohort shrinks — fewer total positions per trial, but a larger fraction of the surviving positions cleanly approach-without-revisit. This tracks the NB22 K-bbox-3 finding (evaluated_rejected per-AOI share: 15.5% → 20.1%).

## Feature coefficients (standardized, full-data refit)

| Feature | Absolute | Organic | Δ |
|---|---|---|---|
| **mean_dist** | **+0.737** | **+0.634** | −0.103 |
| dwell_in_proximity_ms | +0.398 | **+0.470** | +0.072 (strengthens) |
| **direction_changes** | **+0.567** | **+0.320** | **−0.247** (substantial weakening) |
| retreat_dist | +0.252 | +0.257 | ≈ |
| final_dist | +0.237 | +0.257 | ≈ |
| max_approach_velocity | +0.170 | −0.015 | sign flip, near-zero magnitude |
| frac_decreasing | +0.047 | +0.085 | ≈ |
| mean_approach_velocity | −0.051 | +0.082 | sign flip, near-zero magnitude |
| min_dist | −0.139 | −0.005 | toward zero |

**Notable.** `direction_changes` (+0.567 → +0.320) loses substantial weight under bbox. Under absolute attribution, more direction changes meant more "deferred" — cursor wandering as the user fixated other candidates. Under bbox the signal is still positive but ~half as important. **Hypothesis:** under absolute, gap-region cursor drift was registering as "approaches with high direction-change counts" and pulling the deferred class toward those records. Under bbox the gap drift no longer registers as approach, so direction_changes loses some of its proxy power.

`mean_dist` remains the dominant feature in both regimes.

## Implications for AR replay rebuild

1. **Use `m5_final_model_organic.json`** for any rebuild that consumes bbox AOIs. Path: `scripts/output/m5_cursor_only_taxonomy_organic/m5_final_model.json`.
2. **Operating threshold shifts.** If `approach-retreat/scripts/build_replay_trial.py` hardcodes the legacy threshold (0.395), update to 0.489. Better: pull from `m5_final_model.json["youden_j"]`.
3. **Curation captions referencing "5 DEFERRED AOIs" etc. need automated diff.** The class predictions on the curated trials may shift between models even before you account for the underlying NB22 K-bbox-9 (411 click reattributions). Rebuild + diff before re-publishing.
4. **CIKM §5 hard-negative purity claim stands** — 55% FN, no rewrite needed.
